з«¶иіҪ / жҲҗе°ұ

Google Mobile Web Specialist

2019 - 2022пјҢйҖҷжҳҜдёҖй …з”ұ Google Developer дё»жҢҒзҡ„иӘҚиӯүпјҢз”ЁдҫҶжё¬и©ҰиЎҢеӢ•з¶Ій Ғй–ӢзҷјжҠҖиЎ“гҖӮ

жҲ‘зҡ„иӘҚиӯү - https://www.credential.net/xh0j2ccc

DevDays Asia 2017 Chatbot й–ӢзҷјеӨ§иіҪ

Chatbot еңЁз•¶жҷӮжӯЈеңЁиө·жӯҘпјҢжүҖд»ҘжҲ‘еҖ‘е»әз«ӢдәҶдёҖеҖӢжңҚеӢҷи®“дәәеҖ‘еҸҜд»Ҙи—үз”ұз°Ўзҹӯзҡ„е°Қи©ұдҫҶе®үжҺ’д»–еҖ‘зҡ„иЎҢзЁӢпјҢдҪҝз”Ё Google Place зӯүжҷҜй»һиіҮж–ҷдҫҶжәҗгҖӮйӣ–з„¶зҸҫеңЁзңӢдҫҶд»–жҳҜеҖӢзіҹзі•зҡ„жғіжі•пјҢйҖҸйҒҺ Chatbot дјјд№ҺжІ’жңүжҜ”зӣҙжҺҘдҪҝз”Ёз¶Із«ҷдҫҶзҡ„зӣҙиҰәпјҢдҪҶйҖҷд»Қз„¶жҳҜеҖӢиҲҮдҪҝз”ЁиҖ…дә’еӢ•дёҖеҖӢеүөж–°зҡ„ж–№ејҸпјҢжҲ‘еҖ‘жңҖзөӮйҖІе…ҘжұәиіҪгҖӮ

Channel9 йҢ„еҪұ

ж–°еҢ—еёӮжЁӮйҪЎзЁӢејҸй–ӢзҷјеӨ§иіҪ - йҠ…зҚҺ

жҲ‘еҖ‘жғіиҰҒи—үз”ұе…ұдә«дҪҸе®ҝпјҢеўһеҠ е№ҙиј•дәәиҲҮй•·иҖ…зҡ„дә’еӢ•пјҢж–јжҳҜжҸҗеҮәдёҰеҜҰдҪңдәҶеҝҳе№ҙд№ӢдәӨйҖҷеҖӢе№іеҸ°гҖӮдҪҝз”ЁиҖ…еҖӢд»Ҙи—үз”ұдәӨжҸӣеӢһеҠӣгҖҒжҠҖиғҪжҲ–жҲҝеұӢз©әй–“пјҢдҫҶйҒ”еҲ°дә’еҲ©зҡ„зӣ®жЁҷгҖӮ

ж–°иҒһе ұе°Һ Newtalk / з«¶иіҪз°Ўе ұ

2016 HackNTU - Social Design зө„з¬¬дёҖеҗҚ

еңЁзӮәжңҹдёүеӨ©зҡ„вҝҠе®ўжқҫиҲҮеңҳйҡҠзҷјжғідёҰе»әвҪҙиј”е°ҺвјҲеҖ‘дҪҝвҪӨеҗ„зЁ®вјҝж©ҹжҮүвҪӨзЁӢејҸзҡ„е№іеҸ°пјҢзҚІеҫ—з¬¬вјҖеҗҚгҖӮ

з«¶иіҪз°Ўе ұ / Source Code

2015 еҸ°еҢ—еёӮж”ҝеәңгҖҢйӣІз«ҜеҚіжҷӮй–Ӣж”ҫиіҮж–ҷжңҚеӢҷгҖҚз«¶иіҪ - е…ҘеңҚ

еҲ©з”ЁеҸ°еҢ—еёӮж”ҝеәңй–Ӣж”ҫиіҮж–ҷд»ҘеҸҠеҫ®и»ҹзҡ„йӣІз«ҜжңҚеӢҷ (Azure web service, machnine learning studio) пјҢе®ҢжҲҗеҒңи»Ҡе ҙеҚіжҷӮиіҮж–ҷд»ҘеҸҠй җжё¬з©әй–“жҹҘи©ўе№іеҸ°пјҢеңЁз«¶иіҪдёӯзҚІеҫ—е…ҘйҒёгҖӮ

жҸҗдҫӣдҪҝз”ЁиҖ…жҲ–дјҒжҘӯеҲҶдә«ж—…зЁӢзҙҖйҢ„зҡ„е№іеҸ°гҖӮ

жҲ‘еҖ‘еүөйҖ дәҶеҲҶдә«ж—…зЁӢзҡ„еҸҰеӨ–дёҖзЁ®ж–№ејҸпјҢзөҗеҗҲж–Үеӯ—и§ЈиӘӘгҖҒең–зүҮгҖҒеҪұзүҮе’ҢиЎ—жҷҜз•«йқўгҖӮдёҚеҸӘжңүдёҖиҲ¬дҪҝз”ЁиҖ…пјҢе ҙеҹҹе•Ҷе®¶д№ҹеҸҜд»Ҙи—үз”ұжҲ‘еҖ‘зҡ„е№іеҸ°жҺЁеҮәе®ҳж–№зүҲжң¬пјҢи®“жӣҙеӨҡдәәдәҶи§Јд»–еҖ‘зҡ„ж—…йҒҠжҷҜй»һгҖӮ

е№іеҸ°дҪҝз”Ёз…§зүҮзҡ„ EXIF иіҮж–ҷгҖҒGPS зҙҖйҢ„д»ҘеҸҠе…¶д»–жңүз”Ёзҡ„иіҮиЁҠдҫҶеҲӨж–·жӢҚж”қзҡ„дҪҚзҪ®пјҢиҮӘеӢ•е’Ң Goolge Map иЎ—жҷҜжҲ–жҳҜжҲ‘еҖ‘жӢҚж”қзҡ„иЎ—жҷҜе°ҚжҮүпјҢдёҠеӮіз…§зүҮеҫҢеҚіеҸҜз”ўеҮәе°Ҳеұ¬зҡ„зҙҖйҢ„гҖӮ

еҗҲдҪңеӨҘдјҙеҖ‘еҸҜд»Ҙжұәе®ҡзү№е®ҡж—…зЁӢзҡ„дё»йЎҢ (иғҢжҷҜз•«йқўгҖҒдё»йЎҢйЎҸиүІзӯүзӯү)пјҢдёӢйқўжҳҜе…¶дёӯдёҖеҖӢжј”зӨәжЎҲдҫӢ

https://view.adventrip.net/web/hoteldedeer

еҸ°йҗөеҲ—вҫһиіҮиЁҠжҹҘи©ў

2014 - 2019 жҹҘи©ўеҸ°йҗөеҲ—вҫһжҷӮеҲ»зҡ„ Android APPпјҢеҸҜеҚіжҷӮжҲ–йӣўз·ҡжҹҘи©ўпјҢзҚІеҫ— 2 иҗ¬ж¬Ўзҡ„дёӢијүж¬Ўж•ёе’Ң 4.63 жҳҹзҡ„и©•еҲҶгҖӮзӣ®еүҚеҸ°йҗөе®ҳж–№е·Із¶“жңүйҮӢеҮә API д»ҘеҸҠжҮүз”ЁзЁӢејҸпјҢеҶҚйҖҸйҒҺз¬¬дёүж–№зЁӢејҸдҫҶи®ҖеҸ–е·Із¶“жІ’жңүеҝ…иҰҒпјҢжүҖд»Ҙжҡ«жҷӮе°Ү APP дёӢжһ¶гҖӮ
